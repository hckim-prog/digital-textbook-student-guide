# 한빛+ 디지털 교재 수강 학생 안내 페이지

## 프로젝트 목적

2026학년도 2학기 수강 학생에게 한빛+ 디지털 교재의 구매 방법과 이용 흐름을 안내하는 정적 페이지입니다.

학생은 강의계획서 또는 LMS 공지에 안내된 강좌별 전용 구매 링크에서 수강 강좌에 맞는 디지털 교재를 구매합니다.

## 실행 방법

별도 빌드 과정 없이 정적 파일로 실행할 수 있습니다.

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

브라우저에서 `http://127.0.0.1:4173/index.html`을 열어 확인합니다.

## 수정한 주요 섹션

- 히어로 영역을 `수강 학생 안내문 / 2026학년도 2학기` 메시지로 변경
- 1분 영상 섹션 문구를 학생 이용 흐름 중심으로 변경
- 디지털 교재 구성 섹션을 전자책 본문, 강의교안, 쉬운해설과 해설영상, 추가 연습문제, AI 챗봇으로 재구성
- `종이책과 비교해보면` 가격/구성 비교 섹션 추가
- 구매 방법을 강의계획서 또는 LMS 공지의 강좌별 전용 구매 링크 기준으로 정리
- 통합 뷰어와 주요 기능 설명을 학생 학습 경험 중심으로 수정
- 체험 영역을 수업 전 구매 링크와 마이한빛 확인 안내로 변경
- FAQ 전체를 학생 구매/이용 질문으로 교체
- 푸터를 수강 학생 안내 자료 문구로 변경

## 구매 흐름

1. 강의계획서 또는 LMS 공지에서 강좌별 전용 구매 링크를 확인합니다.
2. 전용 구매 링크를 열어 수강 강좌에 맞는 구매 페이지로 이동합니다.
3. 교재명, 포함된 구성, 가격, 환불 규정을 확인하고 결제합니다.
4. 한빛+ 로그인 후 마이한빛에서 구매한 디지털 교재를 확인하고 이용합니다.
